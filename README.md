### Hidden Founders Coding Challenge
### About
A front end web app made with react js to export your facebook photos to firebase.
### Usage
1. install all the necessary dependencies

```
npm install
```
2. hit npm start and you are good to go

```
npm start
```
#### Technologies used
> Reactjs
> Facebook SDK
> Firebase SDK
